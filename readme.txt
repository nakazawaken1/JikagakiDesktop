■概要
デスクトップにマウスやタブレットで直接書き込みができるツールです。

■特徴
・デスクトップ上に透明な紙をおいてその上からマウスやタブレットで自由に線を描くことができます。
・シンプルで直感的に使える操作を重視して作成しました。
・線の色や太さを変えたり、消しゴムで消したり、書いた内容を保存、復元することができます。
・色や太さはワンキーで切り替えることができます。
・ESCキーやメニューで最小化すれば一時的に機能をOFFにできます。
・右クリックで、全ての機能のメニューが表示されます。
・インストール不要で JikagakiDesktop.exe の1ファイルのみで動作します。

■おすすめの使い方
パソコンの画面の説明をするとき、このソフトで書き込みながら説明するとより伝わりやすいでしょう。
また、思いついたことを手書きしたいときにこのソフトで書けば紙を無駄にせず、保存することもできます。

■取り扱い種別
フリーソフト

■動作環境
Microsoft .NET Framework 3.5 以上がインストールされているPC

インストールされていない場合は
http://www.microsoft.com/downloads/details.aspx?familyid=AB99342F-5D1A-413D-8319-81DA479AB0D7&displaylang=ja
よりダウンロード、インストールを行ってください。

■インストール方法
JikagakiDesktop.exeを任意のフォルダに配置してください。

■アンインストール方法
JikagakiDesktop.exeを削除してください。

■実行方法
JikagakiDesktop.exeをダブルクリックするとデスクトップに書き込める状態になります。
終了したい場合は、右クリック→「終了」をクリック、またはAlt+F4を押してください。
